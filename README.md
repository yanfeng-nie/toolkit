# ToolKit

This is a Personal ToolKit, but you can clone and pull request to it to help build a more complete one.

## QEMU

In `./qemu`, I give out three bash files for start-to-end QEMU downloading and installation process. You should run in order:

1. `qemu_download.sh`
2. `qemu_denpendencies_install.sh`
3. `qemu_install.sh`

to complete.
